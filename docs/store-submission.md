# 툭 — 앱스토어 출시 체크리스트 (iOS 우선)

앱 ID: `cloud.tukapp.app` · 표시 이름: **툭** · 라이브: https://tuk-app.vercel.app

## 0. 선행 — Apple 개발자 계정 (님이 시작)

- **developer.apple.com** → Account → 유료 멤버십 등록 (**$99/년**).
- 개인이면 개인 이름으로, 사업자면 D-U-N-S 번호 필요(개인 추천).
- 승인에 보통 **24~48시간**. 이게 있어야 App Store Connect에 앱을 올릴 수 있다.

## 1. App Store Connect에 앱 만들기 (계정 승인 후)

appstoreconnect.apple.com → 나의 앱 → **+ 새로운 앱**
- 플랫폼: iOS · 이름: **툭** · 기본 언어: 한국어
- 번들 ID: `cloud.tukapp.app` (Xcode에서 자동 등록됨) · SKU: `tuk-ios-001`

## 2. 앱 정보 (아래 초안 그대로 붙여넣기)

- **이름**: 툭
- **부제(30자)**: 안 써도 괜찮은 기록
- **카테고리**: 라이프스타일 (2차: 건강 및 피트니스)
- **프로모션 텍스트 / 설명 / 키워드**: 아래 "스토어 문구" 참고
- **지원 URL**: https://tuk-app.vercel.app
- **개인정보 처리방침 URL**: https://tuk-app.vercel.app/privacy

## 3. 앱 개인정보(App Privacy) 답변

수집하는 데이터:
- **이메일 주소** — 계정/로그인용. 사용자에 연결됨. 추적 안 함.
- **사용자 콘텐츠(기록 텍스트·사진)** — 앱 기능 제공용. 사용자에 연결됨. 추적 안 함. 광고에 안 씀.
- 추적(Tracking): **없음**. 서드파티 광고: **없음**(현재). 애널리틱스: **없음**.
- "데이터가 사용자에게 연결됨(Linked)", 용도는 **앱 기능(App Functionality)**만 선택.

## 4. 연령 등급

- 앱 자체 콘텐츠에 성인/폭력 요소 없음 → 대체로 **4+**.
- 웹 접근은 자사 도메인만 로드(무제한 웹 아님).
- 참고: 만 14세 미만 가입 시 앱 내에서 보호자 동의를 받음(국내법 대응).

## 5. 스크린샷 (시뮬레이터에서 캡처)

필요 크기(대표): **6.9"(iPhone 16 Pro Max 등)**, **6.5"** 각각 3~5장.
- 시뮬레이터에서 화면 띄우고 **⌘S**(또는 Device→Save Screen) 로 저장.
- 추천 화면: 온보딩 "안 써도 괜찮아요" / 홈(기록+오늘의 나무) / 나무(한 달) / 알림 칩 / 설정 테마.

## 6. 빌드 업로드 (Xcode)

1. Xcode 상단 기기 선택을 **Any iOS Device (arm64)** 로.
2. **Product → Archive** (몇 분 소요).
3. Organizer 창 → **Distribute App → App Store Connect → Upload**.
   - 서명은 "Automatically manage signing" + 유료 팀 선택이면 자동 처리.
4. 업로드 후 App Store Connect에서 그 빌드를 앱 버전에 선택.

## 7. 제출

- 위 정보 + 스크린샷 + 빌드 채우고 **심사 제출**.
- 첫 심사는 보통 1~3일. 리마인더/알림 권한 사용 사유를 물으면 "사용자가 지정한 시간 알림" 이라고 답.
